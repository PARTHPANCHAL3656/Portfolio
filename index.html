Things I Built.

01
Featured · AI Literacy Platform
MaxPrompt

A gamified AI literacy and prompt engineering platform. Users level up by solving real prompting challenges, unlock Canvas API achievement certificates, and push through a 7-day progressive challenge system that actually teaches them to think in prompts — not just read about it.

Engineered a Vercel serverless proxy to keep API keys secure, integrated OpenRouter for model flexibility, and designed a live level-up detection engine that generates downloadable PNG certificates via Canvas API.
